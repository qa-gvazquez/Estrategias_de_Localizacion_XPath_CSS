# Estrategias_de_Localizacion_XPath_CSS

Este repositorio es para mí el repaso de una habilidad esencial para desempeñarse como QA Automation; diseñar estrategias de localización robustas para utilizar en la creación de frameworks de automatización.

Este repositorio resume lo aprendido en 2 cursos. 

1.- [XPath and CSS Locators for Test Automation (2026)](https://www.udemy.com/course/xpath-locators-for-selenium/) de [Dmitry Shyshkin](https://github.com/dimashyshkin) y 


2.- el libro [Estrategias de Localización de Selenium WebDriver: Guía Práctica para Potenciar las Pruebas Automatizadas](https://www.amazon.com.mx/Estrategias-Localizaci%C3%B3n-Selenium-WebDriver-Automatizadas-ebook/dp/B084MF1NBG) de [Karine Ramos](https://github.com/kramosqa)
